# ZenTool

A modern web application built with Next.js and Radix UI.

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Tech Stack

- Next.js 16
- React 19
- Radix UI Primitives
- Radix UI Themes
- Tailwind CSS
- TypeScript
